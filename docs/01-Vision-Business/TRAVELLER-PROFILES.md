# Purpose

This document defines the primary traveller profiles served by Search My Vacation.

These profiles help guide product design, customer experience, marketing, CRM strategy, AI recommendations, and future business decisions.

Every major feature should improve the experience for one or more traveller profiles defined in this document.

We do not build features for everyone.

We build meaningful experiences for specific types of travellers.

When we deeply understand our travellers, technology becomes simpler, recommendations become better, and trust becomes stronger.

# Traveller Profile 1

## Family Explorer

### Typical Age

30–50 Years

### Travels With

- Spouse
- Children
- Parents (sometimes)

### Typical Budget

₹75,000 – ₹3,00,000

### Travel Frequency

1–2 holidays per year

### Goals

- Spend quality family time
- Create memorable experiences
- Safe destinations
- Comfortable travel
- Good accommodation
- Child-friendly activities

### Pain Points

- Too many confusing packages
- Uncertain about destination suitability
- Limited vacation days
- Budget planning
- Travelling with children

### Opportunities for Search My Vacation

- Travel Confidence
- Trip Scorecard
- Family-friendly itineraries
- Packing checklist
- Personalized recommendations

# Design Principles

Every new feature should answer the following questions.

- Which traveller profile benefits?
- What problem does it solve?
- Does it reduce effort?
- Does it increase trust?
- Does it improve the travel experience?

If a feature cannot clearly answer these questions, it should be reconsidered before development.
