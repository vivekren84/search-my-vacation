# Purpose

Icons enhance usability by providing quick visual recognition of actions, destinations, services, and information.

A consistent icon system improves navigation, reinforces brand identity, and creates a polished user experience.

Icons should support content—not compete with it.

---

# Design Principles

Every icon used throughout SearchMyVacation should be:

- Simple
- Clean
- Modern
- Minimal
- Recognizable
- Consistent

Icons should communicate meaning instantly without requiring additional explanation.

---

# Icon Style

Preferred Style:

- Outline (Line) Icons

Characteristics:

- Rounded corners
- Smooth curves
- Minimal detailing
- Balanced proportions
- Contemporary appearance

Avoid:

- 3D icons
- Cartoon-style icons
- Skeuomorphic icons
- Highly decorative icons
- Emoji-style graphics

---

# Stroke Weight

Use a consistent stroke width throughout the application.

Recommended:

- 2px stroke for standard UI sizes
- Rounded line caps
- Rounded joins

Never mix different stroke weights within the same interface.

---

# Filled vs Outline

Primary UI:

✅ Outline Icons

Filled icons may only be used for:

- Active navigation states
- Selected filters
- Favorite / Wishlist indicators
- Rating stars
- Notifications requiring emphasis

Maintain consistency within each component.

---

# Corner Radius

Preferred appearance:

- Rounded edges
- Friendly geometry

Avoid sharp or aggressive corners.

Rounded icons better reflect the welcoming nature of the SearchMyVacation brand.

---

# Visual Consistency

All icons should share:

- Equal optical weight
- Similar proportions
- Matching corner radius
- Matching stroke width
- Similar padding
- Similar visual balance

Do not combine icons from multiple unrelated icon families.

---

# Recommended Icon Library

Preferred icon sets:

- Lucide Icons (Primary)
- Heroicons (Secondary)
- Phosphor Icons (Optional)

Selection priority:

1. Lucide
2. Heroicons
3. Phosphor

Avoid mixing libraries unless absolutely necessary.

---

# Standard Icon Sizes

Navigation:

24px

Buttons:

18–20px

Cards:

20–24px

Feature Sections:

32px

Large Marketing Sections:

48–64px

Maintain spacing around icons to ensure visual clarity.

---

# Brand Color Usage

Default icon color:

Primary Navy

Interactive states:

- Brand Blue
- Success Green
- Warning Amber
- Error Red

Avoid using multiple colors within a single icon.

Icons should remain visually simple.

# Interactive States

Hover:

- Slight color change
- Smooth transition

Active:

- Filled icon (where applicable)
- Brand Blue highlight

Disabled:

- Reduced opacity
- Lower contrast

Avoid excessive animations.

---

# Icon Animation

Animations should be subtle.

Recommended:

- Fade
- Scale (up to 105%)
- Gentle movement
- Smooth transitions

Avoid:

- Bouncing
- Flashing
- Rotating
- Continuous looping

Animation should reinforce interaction rather than distract.

---

# Accessibility

Icons must never be the only indicator of meaning.

Whenever appropriate:

- Include text labels
- Provide accessible names
- Support screen readers
- Ensure sufficient color contrast

Decorative icons should be marked appropriately for assistive technologies.

---

# Future Scalability

As SearchMyVacation expands, new icons should follow the same design language.

Future modules may include:

- AI Travel Assistant
- Trip Planner
- Rewards
- Loyalty Program
- Corporate Travel
- Community
- Reviews
- Emergency Assistance

All new icons should maintain visual consistency with the established system.

---

# Do's

✓ Use one icon family wherever possible

✓ Maintain consistent stroke width

✓ Use icons to improve usability

✓ Keep icons visually lightweight

✓ Maintain generous spacing

✓ Ensure icons remain recognizable at small sizes

---

# Don'ts

✗ Mix outline and filled styles unnecessarily

✗ Use decorative or overly detailed icons

✗ Stretch or distort icons

✗ Use inconsistent sizes

✗ Replace text labels with icons when clarity would suffer

✗ Over-animate interface elements

---

# Summary

The SearchMyVacation icon system should feel:

- Modern
- Friendly
- Professional
- Lightweight
- Intuitive

A consistent icon language helps users navigate effortlessly while reinforcing the premium, trustworthy personality of the brand.
