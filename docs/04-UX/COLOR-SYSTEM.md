# Purpose

The Color System defines the official colour language for Search My Vacation.

Rather than serving as a simple list of colours, this document establishes the emotional purpose, usage guidelines and reusable design tokens that will be used consistently across the website and future digital products.

The objective is to create a visual identity that communicates trust, warmth, professionalism and memorable travel experiences.

# Color Philosophy

Colour should communicate emotion before information.

Search My Vacation should feel calm, trustworthy and premium while introducing moments of excitement through carefully placed accent colours.

The interface should never appear overly colourful or visually overwhelming.

Whitespace and restraint are considered essential design elements.

---

# Emotional Colour Palette

| Emotion | Colour Role |
|----------|-------------|
| Trust | Primary Blue |
| Adventure | Accent Orange |
| Calm | White |
| Clarity | Light Grey |
| Confidence | Deep Navy |
| Success | Green |
| Attention | Amber |
| Error | Red |

Only the Primary Blue and Accent Orange define the Search My Vacation brand.

Supporting colours exist to improve usability and accessibility rather than branding.

- Trust

- Reliability
- Confidence

Typical Usage:

- Logo

- Links
- Interactive Elements

---

## Accent Colour

Represents:

- Adventure
- Discovery
- Warmth
- Optimism

Typical Usage:

- Primary Calls-to-Action
- Hover States
- Highlights
- Icons
- Small Visual Accents

Accent colours should be used sparingly to preserve visual impact.

---

# Neutral Palette

Neutral colours create visual balance and improve readability.

The interface should primarily consist of neutral colours supported by brand accents.

Typical Neutral Colours:

- White
- Warm White
- Very Light Grey
- Medium Grey
- Charcoal

Typical Usage:

- Backgrounds
- Cards
- Dividers
- Text
- Section Separation

---

# Semantic Colours

Semantic colours communicate system status rather than branding.

| Purpose | Typical Colour |
|----------|----------------|
| Success | Green |
| Information | Blue |
| Warning | Amber |
| Error | Red |

These colours should remain consistent throughout the application.

---

# Colour Usage Guidelines

Approximate visual balance across each page:

- 70% Neutral Colours
- 20% Primary Brand Colour
- 10% Accent Colour

The objective is to create a calm, premium interface with clearly identifiable calls-to-action.

---

# Colour Accessibility

Search My Vacation adopts accessibility as a core quality principle.

All colour combinations shall:

- Meet WCAG AA contrast requirements wherever practical.
- Maintain readability across desktop and mobile devices.
- Avoid using colour alone to communicate important information.
- Preserve usability for users with visual impairments.

Accessibility takes precedence over aesthetic preference.

# Future Colour Validation

The exact colour palette (HEX values) will be finalised during the visual design phase after reviewing multiple colour palette options against:

- Brand Identity
- Accessibility (WCAG AA)
- Visual hierarchy
- Photography
- Typography
- Overall emotional impact

Colour values should not be finalised until validated within realistic interface designs.

---

# Future Evolution

The emotional purpose and usage principles defined in this document are expected to remain stable.

Specific colour values may evolve over time as the product matures, provided they continue supporting the Search My Vacation brand and customer experience.
